shuffleSplit algorithm parameters

(int) Number of re-shuffling & splitting iterations.
(float, int, or None)
  • If float, should be between 0.0 and 1.0 and represent the proportion of the dataset to include in the test split.
  • If int, represents the absolute number of test samples.
  • If None, the value is automatically set to the complement of the train size.
(float, int, or None)
  • If float, should be between 0.0 and 1.0 and represent the proportion of the dataset to include in the train split.
  • If int, represents the absolute number of train samples.
  • If None, the value is automatically set to the complement of the test size.
(int or 'RandomState') Pseudo-random number generator state used for random sampling.
